# StrangerChat Monorepo

Đây là Monorepo của dự án **StrangerChat**. Cấu trúc thư mục được phân bổ như sau:

```text
StrangerChat/
├── backend/          # Source code Golang backend (REST API, gRPC, WebSocket)
├── frontend-user/    # Dự án Next.js 1 dành cho người dùng (Client Web)
├── frontend-admin/   # Dự án Next.js 2 dành cho trang quản trị (Admin Dashboard)
└── .gitignore        # Global gitignore file
```
